# FeatureGuard

Lightweight Redis-based feature-flagging for Ruby apps.

## Installation

Add this line to your application's Gemfile:

    gem 'feature_guard'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install feature_guard
